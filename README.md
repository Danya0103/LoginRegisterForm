# LoginRegisterForm

Simple and clean authentication system built with ASP.NET Core MVC.

## Features
* User registration and login
* Data validation
* Dark-themed minimalist UI
* Database integration via EF Core

## Tech Stack
* **Language:** C#
* **Framework:** ASP.NET Core MVC
* **ORM:** Entity Framework Core
* **Database:** SQL Server

## Setup

1. Clone the repository:
   ```bash
   git clone [https://github.com/Danya0103/LoginRegisterForm.git](https://github.com/Danya0103/LoginRegisterForm.git)

Configure the connection string in appsettings.json.

Update the database:

```bash
dotnet ef database update
```

Run the application:

 ```bash
dotnet run
```
